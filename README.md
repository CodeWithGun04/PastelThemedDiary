# PastelThemedDiary
a soft pastel diary with mood graph and real AI responses
* a soft place to write. no login. no tracking. just you and your words.

## features
- name it after yourself
- mood tracking graph (last 7 days)
- AI that reads what you write
- write as much as you want
- auto date + manual override
- pastel pink/blue/lavender aesthetic

## tech stack
- html5, css3, vanilla javascript
- localstorage api
- canvas api for graph
- vercel for hosting

## live demo
[try it here](https://pastel-themed-diary.vercel.app/)
